- ツール説明
    - このツールは、Jenkins CI専用、デフォルトのWドライブを他のドライブに変更し、複数Branchの同時ビルドを実現するものです。

- 事前準備
    - 「Cドメコン_リプロ簡易手順書.xlsx」の「補足_Pythonのインストール」シートと同様に、Pythonをインストールする
    - 下記Pythonライブラリ(chardet: ファイルエンコードを検知する)をインストールする。　※インストール方法は「補足_Pythonのインストール」シートと同様
        - pip install chardet

- 実行方法
    - コマンドプロンプト or Powershellを実行してください。
    - 「tool\RenameMountDriveInfo」フォルダに移動する。
    - Batchファイルを実行する。
        - 実行方法：x_sb.bat [マウントしたいドライブ]   ※例：　x_sb.bat Y
        - 17ファイルを対象に、文字列置換を実施。
        - 「完了」を出力されたら、実行完了。
        - 指定ドライブにマウント済み。指定ドライブからJenkinsビルドが可能

- 制約事項
    - リポジトリを新規チェックアウト/Clone後、当ツール1回のみ実行可能。
    - 1回チェックアウト/Clone後、複数回実行は想定ユースケース外。
